# Kalkulatori në C++

## Përshkrimi i Projektit
Ky projekt është një kalkulator i avancuar i zhvilluar në C++. 
Ai kryen operacione bazë matematike si:
- Mbledhje
- Zbritje
- Shumëzim
- Pjesëtim
- Fuqi (a^b)
- Rrënja katrore (sqrt)

Projektin e shoqëron edhe një **flowchart** i krijuar me draw.io, i cili tregon logjikën e algoritmit.

---

## Teknologjitë e Përdorura
- C++
- Git & GitHub
- draw.io (Flowchart)
