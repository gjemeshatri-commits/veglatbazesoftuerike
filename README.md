# Kalkulatori në C++

## Përshkrimi i Projektit
Ky projekt është një **kalkulator i avancuar** i zhvilluar në C++.  
Ai kryen operacione matematike të ndryshme, duke përfshirë:

- Mbledhje (+)
- Zbritje (-)
- Shumëzim (*)
- Pjesëtim (/)
- Fuqi (a^b)
- Rrënja katrore (sqrt)

Programi ka **menu interaktive**, **loop për vazhdim**, dhe **validim input-i**, duke parandaluar gabime si pjesëtimi me zero.  

Flowchart-i i krijuar me **draw.io** tregon logjikën e algoritmit dhe vendimet që merr programi gjatë ekzekutimit.
